## Install and Run

```bash
git clone https://github.com/pazoff/Cheshire-Cat-Qt-GUI.git
cd Cheshire-Cat-Qt-GUI/
conda create -n CCatQt5GUI python=3.10
conda activate CCatQt5GUI
pip install -r requirements.txt
python3 CCatQt5GUI.py
